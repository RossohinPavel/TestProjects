Добавление текстового шаблона в программу.

Для добавления текстового шаблона в программу создайте в каталоге /data/MailSamples текстовый документ (в формате txt). 
В файле нужно явно использовать кодировку UTF-8 для правильного отображения кириллических символов. 
Для простоты, можно скопировать текущий файл (__readme.txt), переименовать его и отредактировать текст.

Название файла будет названием шаблона, которое будет отображаться в программе. Название можно давать любое. 
Желательно писать понятные и сжатые названия.

Использование.

Выбираем шаблон в окне и жмем кнопку - Использовать. Текст из шаблона будет помещен в буфер обмена, откуда его можно будет вставить в письмо, мессенджер и т.д.

Текстовые шаблоны поддерживают переменные элементы. Они выделены литералом ?% (знак вопроса и процента в начале и в конце слова).
Для правильности работы шаблона использовать такую комбинацию символов в других целях нежелательно.
 
Если такие элементы есть в шаблоне, то программа предложит вам занести значение вместо этого элемента.
Например: "?%Имя?%, здравствуйте".
Соответсвенно, вместо слова Имя будет подставлено то, которое будет указано в соответствующем поле в программе.

Для добавления переменного элемента выделите слово соответствующим литералом по образцу, указанному в примере.
